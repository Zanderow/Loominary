# Loominary

## Description
This application automates the process of monitoring podcast feeds, generating high-fidelity text transcripts, and organizing them within a structured Google Drive environment. It is designed for researchers and lifelong learners who want to transform passive listening into a searchable personal knowledge base.

## Tech Stack
* Language: Python 3.x
* Data Handling: Pandas (for metadata tracking) & DuckDB (for local caching)
* APIs: Spotify Web API (via Spotipy), Google Drive API v3
* Transcription: OpenAI Whisper / Google Speech-to-Text
